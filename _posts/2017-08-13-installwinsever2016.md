## 安装过程

首先，按照以往的习惯

+ 下载镜像
+ 写入镜像文件
+ boot
+ 安装
+ 安装完成

然后我打开了UltraISO，写入，上机。

然后出现了以下问题：
> Windows 无法打开所需的文件 Sources\install.wim

我慌了，难道下载一晚上的iso是废品？？？

然后各种sha1检验。。。。

终于我在官网上看到了这一句话

> 使用UltraISO等软件刻录镜像时默认使用FAT32文件系统.

这就表明：该系统不支持大于4G的文件，
而Server 2016的安装文件install.wim为5.06G，固安装失败。

于是
按照以前的方法刻录镜像到U盘；

更改U盘文件系统：

进入命令行模式，输入 convert f: /fs:NTFS （F盘为我的U盘所在盘符）

复制Sources目录下的install.wim文件<br>
上电<br>
测试

报错
>windows无法安装所需的文件 文件可能损坏或丢失 0x80070570

然后看到知乎的一篇文章如何用U盘安装windows server 2016？
[windows server 2016安装文件单个超过4G，原FAT32格式不支持](https://www.zhihu.com/question/53790214)

> 可以用Rufus2.11制作U盘安装盘，进行安装。
先选择iso镜像，再选择分区方案，用ntfs格式进行启动盘制作，耐心等待便可大功千成。
