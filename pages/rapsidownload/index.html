系统下载

Raspbian 是官方制作的系统. 从这里下载, 或者使用 NOOBS, 一个用于简单下载并安装 Raspbian 的软件.

  • NOOBS

    一个启动引导软件 - 简易的新手安装系统软件

  • Raspbian

    官方制作的操作系统 - 基于debian

Raspberry桌面 (用于 PC 或者 Mac)

Raspberry用的系统在 PC 或者 Mac上无法运行，用这个镜像. 你可以建立一个live系统, 用虚拟机, 甚至安装在你的电脑上.

Raspberry Pi Desktop 可以在上面调试程序，熟悉桌面系统，熟悉使用linux系统，你可以在购买Raspberrypi之前使用此系统，以打好基础; 这个镜像包括 Python, Scratch, Sonic Pi, Java, 等.

  • Raspberry Pi Desktop

    电脑用的raspberry系统-基于debian

第三方操作系统

由第三方提供的 Raspberry Pi可以使用的操作系统:

  • Ubuntu MATE

    Ubuntu 桌面

  • Snappy Ubuntu Core

    Snappy Ubuntu 开发核心

  • Windows 10 IoT Core

    Windows 10 IoT 物联网系统

  • OSMC

    开源的影音系统

  • LibreELEC

    Kodi 的系统

  • PiNet

    Raspberry Pi 教室教师管理系统

  • RISC OS

    不是Linux 发行系统

  • Weather Station
